## 2. Test Otomasyonu (UI)

---

- Ana sayfa açılır. (www.e-bebek.com)
- Arama çubuğuna "biberon" yazılır.
- Arama yapılır.

- "Sepete Ekle" butonuna tıklanır.
- "Sepeti Gör” butonuna tıklanır.
- Ürünün sepete eklendiği kontrol edilir.
